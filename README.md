## Installation
```bash
npm install
npm run dev
open http://localhost:5173
```

## Releasing
On a build, vite will generate a `dist` folder with the compiled files.
It also includes a `robots.txt` and `sitemap.xml` file for SEO purposes.


```bash
npm run build
```
